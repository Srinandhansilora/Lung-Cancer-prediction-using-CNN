Lung Cancer Detection using CNN
Short Description

This project develops a Convolutional Neural Network (CNN) model to detect lung cancer from medical images. The model analyzes lung images and classifies them into Benign, Malignant, or Normal categories. It helps support early diagnosis by automatically identifying patterns in medical imaging data.

Objective / Problem Statement

Lung cancer is one of the leading causes of cancer-related deaths worldwide, mainly due to late detection. Traditional diagnosis relies on manual interpretation of CT scans or X-rays, which can be time-consuming and prone to human error. The objective of this project is to build an automated deep learning model using CNN to assist in accurate and early detection of lung cancer.

Technologies Used

Python

TensorFlow / Keras

OpenCV

NumPy

Matplotlib

Scikit-learn

Model / Method

A Convolutional Neural Network (CNN) architecture is implemented with multiple convolutional, pooling, and fully connected layers. Image preprocessing techniques such as resizing, normalization, and data augmentation are applied to improve model performance and reduce overfitting.

Results / Accuracy

Training Accuracy: 96.23%

Test Accuracy: 95.57%

The model demonstrates strong performance in classifying lung images into benign, malignant, and normal categories.
